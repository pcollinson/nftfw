# Debian binary packages

This directory contains Debian binary packages. The most recent version is also wrapped in a zip file, so that a fixed link to the current version can be used.

However, directly downloading binary files by looking in the listing and clicking on an entry no longer works. You need to click on the links below, and that will download the raw files for you. Alternatively, click on a file in the listing, then you will get a mostly blank page. To get the file click on the Raw download button.

If you want to use curl to get the binary file, then use this URL prefix.
```
curl https://github.com/pcollinson/nftfw/raw/master/package/NAME
```
