# Debian binary packages

This directory contains Debian binary packages. The most recent version is also wrapped in a zip file, so that a fixed link to the current version can be used.

If you attempt to directly download these files by pointing at the name in the listing above and using a right-click and the Download file option, then it will no longer work. You will get a incorrectly named downloaded HTML file. Left clicking on the link will take to an apparently blamnk page, look in the top line and use the \'Raw\' button to download the file.

This appears to be very fiddly to me - so this page contains direct links to the files, which can be right-clicked and should allow download of the file you are after.
